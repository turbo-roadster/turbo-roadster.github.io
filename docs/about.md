# 关于本站

## 项目简介

AI全栈之路是一个系统化的AI全栈学习平台，旨在帮助开发者从基础到进阶，全面提升AI全栈技术能力。

## 内容特色

### 📚 系统化知识体系

我们将前端知识分为四大模块：

- **基础篇**: HTML、CSS、JavaScript 基础知识
- **进阶篇**: 框架原理、工程化、性能优化
- **面试篇**: 高频面试题型和解答技巧
- **算法篇**: 数据结构与算法

### 🎯 实战导向

每个知识点都配有：

- 清晰的概念解释
- 实用的代码示例
- 常见的应用场景
- 最佳实践建议

### 🔄 持续更新

我们会定期更新内容，紧跟前端技术发展趋势，确保内容的时效性和实用性。

## 技术栈

本站使用以下技术构建：

- **VitePress**: 静态站点生成器
- **Vue 3**: 自定义组件
- **TypeScript**: 类型安全
- **GitHub Pages**: 网站托管

## 使用指南

### 导航使用

- 顶部导航栏可以快速切换不同模块
- 左侧侧边栏展示当前模块的详细目录
- 右侧大纲显示当前页面的章节结构

### 搜索功能

点击顶部的搜索按钮（或按 `/` 键）可以快速搜索全站内容。

### 主题切换

点击顶部的主题切换按钮可以在亮色和暗色主题之间切换。

## 学习建议

::: tip 学习路径
1. **新手**: 从基础篇开始，系统学习 HTML、CSS、JavaScript
2. **进阶**: 学习框架原理和工程化知识
3. **面试**: 查看高频面试题，准备技术面试
4. **提升**: 通过算法篇提升编程能力
:::
