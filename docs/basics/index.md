# 基础篇

欢迎来到前端基础篇！这里将帮助你系统地学习和巩固前端开发的基础知识。

## 学习路径

基础篇涵盖以下核心内容：

### HTML
- 语义化标签
- 表单元素
- HTML5 新特性
- 可访问性

### CSS
- 盒模型
- 布局（Flexbox、Grid）
- 响应式设计
- CSS3 动画

### JavaScript
- 基础语法
- 数据类型
- 函数与作用域
- 异步编程

## 为什么要学习基础

扎实的基础是成为优秀前端开发者的关键。无论技术如何发展，这些基础知识都是不变的核心。

::: tip 学习建议
建议按照 HTML → CSS → JavaScript 的顺序学习，循序渐进，打好基础。
:::
