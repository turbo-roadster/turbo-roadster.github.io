# 面试篇

欢迎来到前端面试篇！这里汇总了前端面试中的高频题型和解答技巧。

## 面试准备

### 高频题型分类

#### JavaScript 面试题
- 闭包与作用域
- 原型与继承
- 异步编程
- ES6+ 新特性

#### CSS 面试题
- 布局实现
- 选择器优先级
- BFC 与 IFC
- 响应式设计

#### 框架面试题
- React 生命周期
- Vue 响应式原理
- 组件通信
- 状态管理

#### 工程化面试题
- 构建工具原理
- 性能优化方案
- 前端安全
- 项目架构

## 面试技巧

::: tip 回答技巧
1. **理解题意**: 先确认面试官的问题，避免答非所问
2. **结构化回答**: 按照"是什么 → 为什么 → 怎么做"的结构回答
3. **举例说明**: 结合实际项目经验，让回答更有说服力
4. **深入浅出**: 既要讲清楚原理，也要注意表达的清晰度
:::

## 常见面试流程

1. **自我介绍**: 简洁明了，突出亮点
2. **技术面试**: 基础知识 + 项目经验 + 算法题
3. **项目讨论**: 深入讨论项目细节和技术选型
4. **反问环节**: 准备几个有深度的问题

## 面试准备清单

- [ ] 复习基础知识（HTML、CSS、JavaScript）
- [ ] 梳理项目经验和技术亮点
- [ ] 练习常见算法题
- [ ] 了解目标公司的技术栈
- [ ] 准备自我介绍和反问问题

::: warning 注意事项
面试不仅考察技术能力，也考察沟通能力和学习能力。保持自信，诚实回答，不懂的地方要坦诚说明。
:::
