## 学习方法

::: tip 刷题建议
1. **循序渐进**: 从简单题开始，逐步提高难度
2. **分类练习**: 按照数据结构和算法类型分类练习
3. **总结归纳**: 每做完一题，总结解题思路和技巧
4. **定期复习**: 定期回顾做过的题目，巩固记忆
:::

## 时间复杂度

理解时间复杂度对于算法学习至关重要：

| 复杂度 | 名称 | 示例 |
|--------|------|------|
| O(1) | 常数 | 数组访问 |
| O(log n) | 对数 | 二分查找 |
| O(n) | 线性 | 遍历数组 |
| O(n log n) | 线性对数 | 快速排序 |
| O(n²) | 平方 | 冒泡排序 |
